Команда для запуска
ansible-playbook useradd.yml

Если подключение к серверу подразумевает ввдо пароля то добавить
--ask-become-pass
и ввести SUDO пароль для подключения к серверу

Пароль для vault: cloud

Создать в директории ssh_key файл id_rsa.pub и поместить туда содержимое публичного ключа.
В useradd.yml прописать путь к файлу с публичным путем

В файле group_vars/local_servers прописать имя пользователя и путь до публичного
ключа, пользователя который имеет доступ до выполнения playbook

В файле hosts.txt прописать ip-адреса серверов
